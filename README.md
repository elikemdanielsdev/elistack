# Elistack - Remix Starter Project

Welcome to the **Elistack**! This repository is built to offer a strong foundation for your next web project. With a focus on interactivity, UI styling, iconography, and Directus CMS integration, this starter kit provides a modern and versatile setup to kickstart your web development with Remix and Directus.

## ✨ Features & Stack

- **[Remix](https://remix.run/)**: A full-stack framework for building fast, responsive web apps. Remix offers server-rendered routes, a seamless API layer, and powerful data loading, all while optimizing for great user experiences and high performance.

- **[Directus](https://directus.io/)**: A real-time, open-source Headless CMS that allows you to manage your content in a database. Directus provides a flexible and scalable solution for storing and retrieving your content, making it easy to build dynamic and responsive web applications.
- **[Framer Motion](https://www.framer.com/motion/)**: A library for animations and interactions. Framer Motion makes it easy to animate components, transitions, and gestures in your app, adding smooth, professional motion to enhance user engagement.

- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework that helps you build custom designs quickly. With pre-defined classes, you can create responsive, visually appealing layouts without writing traditional CSS.

- **[Radix UI](https://www.radix-ui.com/)**: A collection of unstyled, accessible components for React. Radix UI ensures your components are both accessible and customizable, providing a strong foundation for building polished UIs with flexibility.

- **[ShadCN](https://shadcn.dev/)**: A collection of styled, accessible components built on Radix UI and Tailwind CSS. ShadCN combines Radix's accessibility with Tailwind's styling, allowing for a seamless integration of accessible, modern design elements.

- **[Lucide React](https://lucide.dev/docs/lucide-react)**: An open-source icon library for React. Lucide React is designed with SVG-based icons that are easy to scale and style, helping you enhance UI elements with lightweight, customizable icons.

- **[clsx](https://github.com/lukeed/clsx)**: A utility for conditionally joining class names in JavaScript. clsx is small, efficient, and ideal for managing complex class combinations in components.

## 📦 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/elikemdanielsdev/elistack.git
cd elistack
npm install
```

Then start the development server:

```bash
npm run dev
```

This will start your project locally, and you can access it at `http://localhost:3000`.

## 📂 Project Structure

Here's an overview of the main directories and files:

- **`/app`**: Contains all Remix app files (routes, components, styles, etc.)
- **`/public`**: Static assets such as images and fonts.
- **`tailwind.config.js`**: Tailwind CSS configuration for styling.
- **`remix.config.js`**: Remix app configuration.

## 🚀 Key Libraries & Tools

Each library has been added to serve a specific purpose and can be further customized:

- **Remix**: A full-stack framework for building fast, responsive web apps. Remix offers server-rendered routes, a seamless API layer, and powerful data loading, all while optimizing for great user experiences and high performance.
- **Directus SDK**: Enables easy connection to Directus CMS, making content management intuitive and scalable.
- **Framer Motion**: Adds smooth, interactive animations that improve the user experience.
- **Tailwind CSS**: Empowers rapid styling with utility classes to streamline your design workflow.
- **Radix UI**: Ensures accessible, unstyled components that can be fully customized to your design needs.
- **ShadCN**: Provides Radix-based components styled with Tailwind CSS, perfect for a cohesive and modern UI.
- **Lucide React**: A versatile icon set to visually enhance your app.
- **clsx**: Helps manage dynamic class names in a clear and organized way.
